Category: Data Visualization

Prompt style: Casual

Title: Real-Time Data Streaming Chart

Prompt: Live updating line chart streaming random data points every second. Shows last 60 seconds of data with smooth scrolling. Multiple metrics tracked simultaneously (CPU, Memory, Network) with separate lines and Y-axes. Toggle metrics on/off. Pause/resume button. Speed control (1x, 2x, 5x). Alert thresholds show red zones on chart. When metric crosses threshold, flash notification and play sound. Stats panel shows current, average, min, max for each metric. Sparklines for each metric in stats. Download last 5 minutes data. Chart auto-scales Y-axis as values change. Make it look like a monitoring dashboard.

Required libraries: react, tailwindcss, recharts, lucide-react, framer-motion
