Title: Quarterly Sales Performance Multi-Ring Chart

Prompt: I need to visualize quarterly sales performance for three products using concentric rings with 3D raised depth effects - inner ring for Q1, middle for Q2, outer for Q3. Each ring divides into segments where each product's arc size shows its share of that quarter's sales. The three products should align across all rings. Color segments green with a pulsing glow animation if sales went up from the previous quarter, red with pulsing glow if down, and gray for Q1. Draw thin curved connector lines from each segment to a performance summary panel positioned directly below the chart showing each product's Q1-to-Q3 growth and which had the most steady performance. Position a legend exactly below the performance summary explaining the color scheme. Preload with sample quarterly sales data.

Required libraries: react, tailwindcss, lucide-react, recharts
