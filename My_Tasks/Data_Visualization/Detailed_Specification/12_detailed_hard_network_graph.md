Category: Data Visualization

Prompt style: Detailed Specification

Title: Network Graph Visualization

Prompt: Create an interactive network graph showing relationships between 20 nodes. Nodes sized by connection count. Connections weighted by interaction strength (line thickness). Force-directed layout with physics simulation. Drag nodes to reposition. Hover shows node name and connection count. Click node to highlight its connections and dim others. Filter nodes by type using checkboxes. Search bar finds and centers on node. Zoom and pan. Display metrics panel showing total nodes, connections, density, and clusters detected. Color nodes by community detection algorithm. Toggle between force and circular layouts. Export graph as JSON.

Required libraries: react, tailwindcss, react-force-graph, lucide-react
