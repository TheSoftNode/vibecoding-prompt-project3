Category: Data Visualization

Prompt style: Feature List

Title: Choropleth Map Visualization

Prompt: Build an interactive US state map colored by data values. Upload CSV with state names and values. Parse CSV client-side. Color states from light yellow (low) to dark red (high) based on value quantiles. Display legend with color scale and range labels. Hover shows state name, value, and rank. Click state to zoom and show detailed stats panel. Pan and zoom map with mouse. Dropdown selects data column if CSV has multiple metrics. Reset zoom button. Export map as SVG. Responsive sizing. Pre-load sample CSV with population data for all 50 states.

Required libraries: react, tailwindcss, d3, papaparse, lucide-react
