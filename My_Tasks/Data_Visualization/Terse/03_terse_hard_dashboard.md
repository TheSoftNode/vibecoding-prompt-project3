Category: Data Visualization

Prompt style: Terse

Title: Interactive Analytics Dashboard

Prompt: Dashboard with 4 chart types: line chart (traffic over time), bar chart (top products), pie chart (user demographics), and table (recent orders). All charts share date range picker affecting displayed data. Real-time update simulation with refresh button. Export each chart as PNG. Click any data point to filter other charts. Metrics cards show total users, revenue, conversion rate, avg order value with sparklines. Responsive 2x2 grid layout. Pre-load with sample e-commerce data for last 30 days.

Required libraries: react, tailwindcss, recharts, lucide-react, date-fns, html2canvas
