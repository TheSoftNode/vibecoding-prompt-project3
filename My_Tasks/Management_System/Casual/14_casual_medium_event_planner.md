Title: Event Planning Manager

Prompt: Manage events easily. Event card shows name, date, time, location, guest count, budget, status (Planning, Confirmed, Completed). Add event form. Task checklist per event (Venue, Catering, Invites, etc) - check off completed tasks. Show progress bar based on completed tasks. Guest list per event - add guests with RSVP status (Pending, Yes, No). Count confirmed guests. Budget tracker shows spent vs total with remaining. Calendar view displays all events by date. Upcoming events section shows next 3. Overdue tasks highlighted red. Make it clean and visual.

Required libraries: react, tailwindcss, lucide-react, date-fns, recharts
