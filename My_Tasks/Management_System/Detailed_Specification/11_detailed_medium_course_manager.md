Category: Management System

Prompt style: Detailed Specification

Title: Course Management Dashboard

Prompt: Create course management system. Course card shows title, instructor, start date, duration weeks, enrolled students, capacity, status (Upcoming, Active, Completed). Add course form with all fields. Enrollment system: click Enroll, select student from list, add to course. Display enrolled students per course. Course detail view shows: description, syllabus topics list, enrolled students table with progress percentage. Mark attendance button records present/absent per session. Calculate attendance rate. Filter courses: by status, by instructor. Search by title. Metrics: total courses, active courses, total enrollments, average class size. Pre-load with 3 courses and 10 students.

Required libraries: react, tailwindcss, lucide-react, date-fns
