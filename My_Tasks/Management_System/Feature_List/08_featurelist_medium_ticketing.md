Title: Support Ticket System

Prompt: Build support ticketing system. Ticket has: title, description, priority (Low, Medium, High, Critical), status (Open, In Progress, Resolved, Closed), assigned agent, created date. Create ticket form. Ticket list view with color-coded priority badges. Filter: by status, by priority, by agent. Sort by date or priority. Click ticket to view details. Status dropdown changes ticket. Add comment to ticket with timestamp. Comments thread shows all updates. Metrics dashboard: open count, resolved today, average resolution time, tickets by priority breakdown. Agent dropdown shows team members.

Required libraries: react, tailwindcss, lucide-react, date-fns
