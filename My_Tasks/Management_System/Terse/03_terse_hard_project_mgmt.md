Category: Management System

Prompt style: Terse

Title: Advanced Project Management

Prompt: Project management with projects, milestones, tasks, team members. Project dashboard shows active projects with progress bars. Each project has milestones (phases with dates). Milestones contain tasks with assignee, status, priority, due date, time estimate. Gantt chart view shows tasks timeline. Kanban board view with columns: Backlog, Todo, In Progress, Review, Done. Drag tasks between columns. Task dependencies: blocking/blocked by relationships shown with arrows. Team member workload view shows assigned tasks and hours. Burndown chart tracks remaining work. Calendar view with task due dates. Time tracking: start/stop timer per task. Filter tasks: by project, by assignee, by status. Overdue tasks alert panel. Dashboard metrics: total projects, active tasks, completed this week, at-risk tasks (overdue or approaching deadline). Export project report as CSV.

Required libraries: react, tailwindcss, lucide-react, date-fns, recharts, react-beautiful-dnd
