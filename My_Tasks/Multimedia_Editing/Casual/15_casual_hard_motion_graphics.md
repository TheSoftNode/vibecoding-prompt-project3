Title: Motion Graphics Studio

Prompt: Build motion graphics tool. Canvas with animation timeline. Add elements: shapes (circle, square, polygon), text, images. Keyframe animation system - set position, scale, rotation, opacity at different times. Easing curves for smooth motion. Path animation: draw path, element follows. Particle effects: confetti, sparkles, smoke. Shape morphing between keyframes. Mask layers. Blur and effects. Looping animations. Preview at different framerates. Export as GIF, video, or Lottie JSON. Library of pre-made animations. Snap to beats if music uploaded. Duration scaling. Group elements to animate together. Layer parenting: child follows parent transforms. Make it powerful but intuitive.

Required libraries: react, tailwindcss, lucide-react, framer-motion, lottie-web
