Title: Video Subtitle Creator

Prompt: Create subtitle editor for videos. Upload video file. Timeline shows video duration. Add subtitle entry at current time. Each subtitle has: text, start time, end time, position (top/center/bottom). Edit subtitle text inline. Drag subtitle bar on timeline to adjust timing. Split subtitle into two. Merge adjacent subtitles. Keyboard shortcuts: spacebar play/pause, left/right arrow frame by frame. Waveform visualization helps timing. Auto-detect speech pauses suggests subtitle breaks. Font customization: family, size, color, background, outline. Preview subtitle rendering over video. Import SRT or VTT files. Export subtitles as SRT, VTT, or burned into video. Translation mode: side-by-side original and translated text. Character per second warning for readability. Search/replace in all subtitles. Spelling checker. Timeline zoom levels. Export timestamps as CSV.

Required libraries: react, tailwindcss, lucide-react, wavesurfer.js
