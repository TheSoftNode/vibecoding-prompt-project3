Title: Photoshop-Style Layer Editor

Prompt: Multi-layer image editor. Canvas with layer stack. Add layers: image, text, shape. Drag layers to reorder in stack panel. Each layer has: opacity slider, blend mode dropdown (Normal, Multiply, Screen, Overlay), visible toggle, lock toggle. Transform layer: move, scale, rotate with handles. Layer effects: drop shadow (offset, blur, opacity), glow, stroke. Crop canvas. Background layer or transparent. Merge layers down. Duplicate layer. Flatten all layers. Group layers into folders. Layer masks for transparency. Adjustment layers: brightness/contrast, hue/saturation. Undo/redo with history panel showing 20 steps. Export flattened image or PSD-like JSON. Keyboard shortcuts.

Required libraries: react, tailwindcss, lucide-react, html2canvas, fabric.js
