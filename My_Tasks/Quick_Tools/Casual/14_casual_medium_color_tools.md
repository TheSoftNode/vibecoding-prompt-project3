Title: Color Palette Generator

Prompt: All-in-one color tool. Color picker shows hex, RGB, HSL. Generate palette: complementary, analogous, triadic, monochromatic. Random palette button. Extract colors from uploaded image (5 dominant colors). Gradient generator with 2-5 color stops, angle control, CSS code output. Contrast checker for accessibility (WCAG AA/AAA). Color blindness simulator. Palette library saves favorites. Export palette as PNG, CSS, JSON, or Adobe ASE. Shade generator creates 10 tints/shades of color. Name that color feature shows color name. Make it designer-friendly.

Required libraries: react, tailwindcss, lucide-react, colorthief
