Title: Developer Multi-Tool Dashboard

Prompt: Ultimate dev dashboard with widgets. JSON formatter, Base64 encoder/decoder, URL encoder/decoder, Hash generator (MD5, SHA256), JWT decoder with claims viewer, Unix timestamp converter with timezone. Diff checker shows text differences side-by-side. Cron expression builder with next run times. Lorem ipsum generator with paragraphs/words count. UUID generator (v4). Color converter between hex/RGB/HSL. Image to Base64 converter. Regex tester mini version. Each tool is a card on dashboard. Drag to rearrange. Minimize/maximize widgets. Dark/light theme. Recent tools section. Favorites star button. Search tools. Export all tool outputs. Make it fast and look like a pro developer would use it daily.

Required libraries: react, tailwindcss, lucide-react, react-grid-layout, crypto-js
