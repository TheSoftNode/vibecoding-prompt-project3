Prompt style: Conversational

Title: REST API Testing Tool

Prompt: I need a Postman-like API tester. URL input with method dropdown (GET, POST, PUT, DELETE, PATCH). Headers editor: key-value pairs, add/remove rows. Request body editor with tabs: JSON, Form Data, Raw Text. Query parameters builder. Authentication section: Bearer token, API key, Basic Auth. Send button makes request, shows loading state. Response panel: status code, headers, body with JSON formatting. Response time displayed. Save requests to collection organized in folders. Environment variables with {{variable}} syntax substitution. Pre-request scripts field. Tests/assertions: status code checks, response time validation, JSON path verification. History shows last 50 requests. Export collection as JSON. Import cURL commands. Code snippet generator for fetch/axios. Dark theme. Keyboard shortcuts.

Required libraries: react, tailwindcss, lucide-react, react-json-view, axios
