Title: JSON Formatter and Validator

Prompt: Create JSON tool suite. Input area for raw JSON text. Format button beautifies with indentation. Minify button compresses to single line. Validate button checks syntax and shows errors with line numbers. Syntax highlighting in output. Collapsible tree view of JSON structure. Search and filter keys. Convert to other formats: CSV, XML, YAML. Diff mode compares two JSON objects showing differences highlighted. Path finder: click any value, shows JSON path. Escape/unescape strings. Sort keys alphabetically. Remove duplicates. Copy formatted output. Error explanation panel for common JSON mistakes. Pre-load with sample API response JSON.

Required libraries: react, tailwindcss, lucide-react, react-json-view
