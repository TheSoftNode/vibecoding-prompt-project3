Title: SQL Query Playground

Prompt: Create SQL learning environment. Load sample database with 4 tables: users, orders, products, reviews. Schema viewer shows table structures with column types and relationships. SQL editor with syntax highlighting, auto-complete for table/column names. Run query button executes against in-memory SQLite. Results display in table with pagination. Export results as CSV or JSON. Query history saves last 20 queries. Execution plan visualizer shows query performance. Common queries library: joins, aggregations, subqueries. Error messages explain SQL issues. Visual query builder for beginners: drag tables, select columns, add conditions. Generated SQL shown. Data visualization: auto-suggest chart types based on query results. Transaction support: begin, commit, rollback. Inject custom data: CSV upload creates temp table. Formatter beautifies SQL. Share query as URL. Keyboard shortcuts: Ctrl+Enter runs query. Pre-load example queries for each SQL concept.

Required libraries: react, tailwindcss, lucide-react, sql.js, react-syntax-highlighter, recharts
