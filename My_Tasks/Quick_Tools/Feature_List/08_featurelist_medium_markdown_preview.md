Title: Markdown Live Editor

Prompt: Build markdown editor with live preview. Split view: editor on left, preview on right. Editor has syntax highlighting for markdown. Preview renders HTML in real-time. Support: headers, bold, italic, lists, links, images, code blocks, blockquotes, tables. Toolbar buttons insert markdown syntax. Word and character count. Export as HTML or markdown file. Copy HTML button. Full-screen mode. Dark mode toggle. Auto-save to localStorage. Template selector with common markdown structures. Pre-load with sample markdown document.

Required libraries: react, tailwindcss, lucide-react, react-markdown, react-syntax-highlighter
