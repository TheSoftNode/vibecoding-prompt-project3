Title: CSV/Excel Data Processor

Prompt: Build comprehensive data processing tool. Upload CSV or Excel file. Display data in editable table. Column operations: rename, delete, reorder by dragging, add calculated column with formula. Row operations: filter with multiple conditions, sort by column, delete rows. Find and replace across table. Merge/split columns with delimiter. Remove duplicates. Data validation: flag errors, invalid dates, out-of-range numbers. Statistics panel per column: count, unique, min, max, average, median. Pivot table builder. Chart generator from selected columns. Export to CSV, Excel, JSON. SQL query builder to filter data. Undo/redo for all operations. Auto-detect column types (text, number, date, boolean). Date formatting options. Number formatting: decimals, thousands separator, currency. Pre-load with sample sales data (100 rows, 8 columns).

Required libraries: react, tailwindcss, lucide-react, recharts, papaparse, xlsx
