Title: Regex Pattern Tester

Prompt: Regex testing tool with pattern input and test string textarea. Match results highlight in text with colors. Capture groups shown separately. Global, case-insensitive, multiline flags. Real-time match count. Match details panel shows start/end positions, matched text, groups. Replace mode with replacement string, preview before/replace all. Common regex library: email, URL, phone, date formats. Regex explainer breaks down pattern with plain English. Test string generator creates sample data matching pattern. Performance timer shows execution time. History saves last 10 patterns. Export matches as JSON or CSV. Code generator outputs regex for JavaScript, Python, PHP. Error messages explain regex syntax issues.

Required libraries: react, tailwindcss, lucide-react, react-syntax-highlighter
