Category: SVG Generation

Prompt style: Conversational

Title: Infographic Chart Builder

Prompt: I need to build SVG infographic elements for a presentation. I want to add two types of elements using buttons: "Add Stat Box" creates a box with editable number and label text fields that render as SVG, and "Add Progress Circle" creates a circle where I can set percentage with a slider that updates the colored arc. Let me drag any element to reposition it anywhere on the canvas. When I select an element, show a color picker to change its color. At the bottom, include an "Export SVG" button that downloads only the SVG elements (not the controls) as an SVG file with their exact positions preserved.

Required libraries: react, tailwindcss, lucide-react, framer-motion

## Justification

The infographic builder works exactly as expected with dynamic SVG element creation. Two add buttons are displayed for creating stat boxes and progress circles. Clicking "Add Stat Box" creates a new stat box with editable text fields that render as SVG with the entered number and label. Clicking "Add Progress Circle" creates a circle with a percentage slider that calculates and updates the colored arc path coordinates in real-time as the slider changes. Every element can be dragged to any position on the canvas, and the new position is preserved when the drag ends. Clicking an element selects it, and when an element is selected, a color picker appears to change its color. At the bottom, the "Export SVG" button downloads only the SVG elements with their exact positions preserved, excluding all controls and UI elements.
