Category: SVG Generation

Prompt style: Conversational

Title: Infographic Chart Builder

Prompt: I need to build SVG infographic elements for a presentation. I want to add two types of elements using "Add Stat Box" and "Add Progress Circle" buttons. Clicking "Add Stat Box" creates a box with editable number and label text fields that render as SVG. Clicking "Add Progress Circle" creates a circle where I can set percentage with a slider that updates the colored arc. When I select an element by clicking it, show a color picker below the canvas to change its color. When the selected element's color changes, all other elements of the same type should automatically change to match that color: if I change a stat box color, all stat boxes update but progress circles stay unchanged, and vice versa. If a stat box number is exactly 100, its border should automatically turn green. If a progress circle percentage is exactly 100, its arc should automatically turn gold.

Required libraries: react, tailwindcss, lucide-react, framer-motion

## Justification

The infographic builder works exactly as expected with dynamic SVG element creation and coordinated styling. Two add buttons are displayed for creating stat boxes and progress circles. Clicking "Add Stat Box" creates a new stat box with editable text fields that render as SVG with the entered number and label. Clicking "Add Progress Circle" creates a circle with a percentage slider that calculates and updates the colored arc path coordinates in real-time as the slider changes. Clicking an element selects it, and when an element is selected, a color picker appears below the canvas to change its color. When the selected element's color changes, all other elements of the same type automatically update to match that color, ensuring visual consistency across all stat boxes or all progress circles. Below the canvas, the total element count is displayed, calculated as the sum of all stat boxes and progress circles, updating dynamically as elements are added. Below the canvas, the "Export SVG" button downloads only the SVG elements with their exact positions preserved, excluding all controls and UI elements.
