Category: SVG Generation

Prompt style: Conversational

Title: Infographic Chart Builder

Prompt: I need to build SVG infographic elements. Add two types: "Add Stat Box" creates a box with editable number and label fields rendering as SVG. "Add Progress Circle" creates a circle with percentage slider that updates the colored arc. Clicking an element selects it and shows a color picker below canvas. When color changes, all elements of the same type update to match. Every 3rd element added (3rd, 6th, 9th, etc.) should pulse with a scale animation continuously. Track total value: sum of all stat box numbers plus all progress circle percentages. Display this total below canvas, updating live as numbers or percentages change. If total value exceeds 500, all stat boxes turn red borders and all progress circles turn red arcs. If any stat box shows exactly 100, its border turns green (overriding red if total > 500). If any progress circle shows exactly 100%, its arc turns gold (overriding red if total > 500).

Required libraries: react, tailwindcss, lucide-react, framer-motion

## Justification

The infographic builder works exactly as expected with layered complexity through cross-feature dependencies, evolving state, and priority-based conditional styling. Two add buttons create stat boxes with editable number/label fields and progress circles with percentage sliders that calculate SVG arc paths. Element creation order is tracked, and every 3rd element added (3rd, 6th, 9th) continuously pulses with scale animation. Clicking an element selects it and shows a color picker below the canvas. Changing color updates all elements of the same type to match. A running total value is calculated by summing all stat box numbers plus all progress circle percentages, displayed below canvas and updating live whenever any number or percentage changes. When total exceeds 500, all stat box borders and progress circle arcs turn red. Individual exact-value conditions override this: any stat box showing exactly 100 turns green border (overriding red), and any progress circle showing exactly 100% turns gold arc (overriding red). Multiple systems interact simultaneously: creation order drives animation, value changes drive total recalculation, total threshold drives group styling, exact values drive individual overrides.
