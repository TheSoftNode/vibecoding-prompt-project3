Title: Wave Interference Patterns

Prompt: 2D wave simulation on water surface. Click to create wave sources. Multiple sources create interference patterns. Toggle between circular waves, plane waves, standing waves. Adjust frequency, amplitude, wavelength per source. Display constructive (bright) and destructive (dark) interference. Phase difference visualization. Damping at edges. Doppler effect when dragging sources. Export wave pattern as image. Slow motion control.

Required libraries: react, tailwindcss, lucide-react
