Title: Restaurant Ordering System with Combos

Prompt: Restaurant ordering system with smart combo deals. Show menu with categories (Appetizers, Mains, Desserts, Drinks), each item has name, price, prep time in minutes, and dietary tags (Vegan, Gluten-Free, Spicy). Click item to add to cart with quantity selector. Cart sidebar on right shows items, quantities, price per item, and running subtotal. Items with Spicy tag show red chili icon. Add modifiers modal when adding items: cooking level for Mains (Rare, Medium, Well Done), size for Drinks (Small, Medium, Large). Modifier choices show in cart like "Burger - Medium". Calculate 10% service charge and 8% tax on subtotal. Auto-detect combo deals: if cart contains 1 Main + 1 Drink + 1 Dessert, apply 15% combo discount (overrides $50 threshold discount if higher, show which discount applied). Display "Combo Deal! You saved $X" badge when combo active. Highlight combo-eligible items in cart with green border. Calculate total prep time as sum of all item prep times, display "Est. Ready in X minutes" at cart top. If cart has Spicy item + Large drink, show "Free Size Upgrade!" notification. Checkout button disabled until cart has items. Clicking checkout shows order summary page with table number input, all details, charges breakdown showing which discount applied, and final total. Confirm button creates order confirmation with order number and prep time estimate. On load, show menu with 3 items per category (include prep times), cart empty.

Required libraries: react, tailwindcss, lucide-react, framer-motion
