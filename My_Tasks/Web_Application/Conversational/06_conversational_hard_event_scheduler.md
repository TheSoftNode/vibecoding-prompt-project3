Category: Web Application

Prompt style: Conversational

Title: Weekly Event Scheduler with Meeting Load

Prompt: I need a weekly calendar view showing Monday through Friday with meeting load tracking. Let me add events by clicking a time slot, which opens a form to enter event title, start time, end time, priority (High, Medium, Low), and recurrence option (None, Daily, Weekly). Display events as colored blocks: red for High, yellow for Medium, green for Low. Each block should span from start time to end time and show the event title. Recurring events automatically create copies on applicable days. Calculate meeting load per day as total hours booked ÷ 8 hours standard workday, display percentage at top of each day column. Turn day column header red when meeting load exceeds 75% (>6 hours booked). If I try to add an event that overlaps with a higher or equal priority event, reject it and show an error message. If it overlaps with lower priority events, replace those events and move them to a "Conflicts" sidebar on the right. The sidebar should show removed events with a "Restore" button for each. Clicking Restore puts the event back if its time slot is now free, otherwise show error with 3 suggested alternative time slots on same day. On load, display the calendar with three sample events: "Team Meeting" Monday 9 AM-10 AM High priority Daily recurrence, "Lunch Break" Wednesday 12 PM-1 PM Medium priority Weekly recurrence, "Code Review" Friday 2 PM-3 PM High priority No recurrence.

Required libraries: react, tailwindcss, date-fns, lucide-react, framer-motion
