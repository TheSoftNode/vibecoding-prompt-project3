Title: Interactive Quiz Application with Timer

Prompt: Build a multi-question quiz application that displays one question at a time with a 30-second countdown timer. Each question should show the question text, countdown timer, four multiple-choice answers as clickable buttons, and a Next button to proceed to the following question. Timer text turns red when less than 5 seconds remain. When a user selects an answer, highlight the selected button in light blue and enable the Next button (which should be disabled until an answer is selected). If timer reaches zero before answering, automatically mark as incorrect and move to next question. Award bonus points: base 10 points per correct answer, plus 5 bonus points if answered in under 10 seconds. After all questions are answered, display a results screen showing the user's total score with breakdown (base points + bonus points), score as a percentage, the number of correct answers out of total questions, average time per question, fastest question time, and a list of all questions with the user's answer, correct answer, and time taken side by side. Questions marked incorrect should have red text, while correct answers should have green text. Questions answered in under 10 seconds should have gold highlighting. Include a Restart button on the results screen that returns to the first question and resets all answers and timers. On initial load, display the first question with pre-loaded quiz data containing three questions: Question 1 about React hooks with correct answer "useState", Question 2 about JavaScript promises with correct answer "then()", and Question 3 about CSS flexbox with correct answer "flex-direction".

Required libraries: react, tailwindcss, lucide-react, framer-motion
