Title: Project Task Tracker with Dependencies

Prompt: Create a project management application that organizes tasks into three projects, each displayed as a collapsible section with a project name header and task count badge. Clicking a project header toggles the visibility of its task list. Each task displays its name, assigned team member from a dropdown (Alice, Bob, Charlie, None), status dropdown (Not Started, In Progress, Completed), and estimated hours input. Tasks also have a priority indicator (High, Medium, Low) shown as a colored dot: red for High, yellow for Medium, green for Low. Include a Dependencies section where users can define that one task blocks another by selecting a blocking task and a blocked task from dropdowns, then clicking Add Dependency. Display all dependencies as a list showing "Task X blocks Task Y" with a Remove button for each. When a task's status is changed to Completed, check if it was blocking other tasks. If so, automatically change those blocked tasks' status from Not Started to In Progress and show a notification banner at the top saying "Task X completion unblocked Y tasks" where Y is the count. The notification should auto-dismiss after 3 seconds. Include a Summary Panel on the right side showing: total tasks across all projects, percentage of completed tasks, total estimated hours, hours completed (sum of hours for completed tasks), and hours remaining. All summary metrics update immediately when task data changes. On initial load, display three projects pre-populated with sample data: "Website Redesign" with 4 tasks, "Mobile App" with 3 tasks, and "Marketing Campaign" with 2 tasks. Pre-populate dependencies showing "Design Mockups blocks Frontend Development" and "API Integration blocks Mobile Testing".

Required libraries: react, tailwindcss, lucide-react, framer-motion
