Category: Web Application

Title: Recipe Manager with Ratings

Prompt: Create a recipe manager with two views: Recipe List and Add Recipe. Users can switch between views using navigation buttons. Recipe List displays all recipes as cards showing recipe name, category, prep time, and star rating (1-5 stars). Display average rating at the top of Recipe List. Add filter dropdowns for category and minimum rating. Each card has a View button that shows full recipe details in a modal overlay including name, category, prep time, star rating, ingredients list, and instructions. The modal should have a close button and a Rate button that allows changing the star rating. Add Recipe view contains a form with name, category dropdown (Breakfast, Lunch, Dinner, Dessert), prep time input, star rating selector, ingredients text area, and instructions text area. A Save button adds the recipe and switches to Recipe List. Display "Recently Viewed" section showing last 3 viewed recipes with timestamps. Filter selections persist when switching views. On initial load, display Recipe List view with three sample recipes: "Pancakes" Breakfast 15 minutes 4 stars, "Caesar Salad" Lunch 10 minutes 5 stars, "Chocolate Cake" Dessert 45 minutes 3 stars.

Required libraries: react, tailwindcss, lucide-react, framer-motion
