Category: Web Application

Prompt style: Feature List

Title: Workflow Builder with Step Dependencies

Prompt: Build a multi-page workflow application with navigation between three views: a dashboard, a Steps view, and a Phases view. The dashboard includes a Step Name text input, a Phase dropdown with options "Phase 1," "Phase 2," and "Phase 3," and a form submission button. If the user submits without a step name, shake the input field and outline it in red until the user begins typing. In the Steps view, display submitted steps as cards sorted by phase order. Each card has a Remove button that deletes the card and immediately updates the step count shown in the Phases view. In the Phases view, display a tree diagram with a "Project" parent node connected to the three phase nodes. Clicking a phase node filters the Steps view to show only steps belonging to that phase.

Required libraries: react, tailwindcss, lucide-react, react-beautiful-dnd
