Rubric Evaluation
Once your prompt and rubrics are ready, it's time to put them to the test. Follow these steps to evaluate the model's output and finalize your task.

Step 1: Generate the Application
Click the Generate App button to trigger the model and create the first version of the application.

Step 2: Access the Application URL
Copy & Paste: Locate the Generate App URL button. You must manually select and copy the entire URL.
⚠️ Note: Clicking the link directly from the interface does not work; it must be pasted into a new browser tab.

Step 3: Capture the Initial Render
Screenshot: Take a screenshot of the application exactly as it appears on the initial load/render.
Upload: You only need to provide one screenshot for the entire task.
Requirement: This screenshot is the primary artifact used to verify visual and layout-based rubric items.

Step 4: Grade and Validate the Solution
Click the Evaluate button to run the rubric checks against the generated output.
Handling Failures: It is perfectly fine if some rubrics fail! You are not required to make every item pass, but you must ensure the failures are legitimate.
Fairness Check: Double-check if your rubrics are fair. A fair rubric accurately reflects the prompt's requirements without being impossible for the model to achieve.
Verification: Manually verify the rubrics that passed. If a rubric passes but the feature clearly doesn't work in the app, the evaluation is inaccurate.
Error Handling:
If your rubrics are unfair: Fix the criteria and restart the entire evaluation process.
